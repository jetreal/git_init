Зачем нужна калибровка?
Сейчас программа использует приблизительные векторы влияния пигментов. Калибровка заменяет их на реальные числа ваших пигментов.

Результат: программа будет точно знать, сколько граммов добавить, чтобы исправить дельту — в одну попытку, без повторных выкрасов.
  1. Нанесите выкрас: 200 г чистой базы + 1 г пигмента.
  2. Замерьте спектрофотометром dL, da, db относительно чистой базы.
  3. Введите значения ниже — программа рассчитает точный вектор.
  4. Повторите для ключевых пигментов (10–15 шт. достаточно).
