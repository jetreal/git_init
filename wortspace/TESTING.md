# 🧪 Nason Lab — Руководство по тестированию

## Как запустить тесты

### Способ 1: Консоль браузера (рекомендуется)
1. Откройте `index.html` в браузере
2. Откройте DevTools (F12) → вкладка Console
3. Введите команду:
   ```javascript
   runAllTests()
   ```
4. Нажмите Enter и смотрите результаты

### Способ 2: Автоматический запуск
В консоли браузера выполните:
```javascript
// Автоматический запуск при загрузке страницы
document.addEventListener('DOMContentLoaded', () => {
    setTimeout(() => runAllTests(), 1000);
});
```

## Интерпретация результатов

### Цвет в допуске
- Все дельты < 0.3 → ✅ Корректировка не требуется

### Корректировка требуется
- dL > 0: Слишком светло → добавить чёрный/тёмный
- dL < 0: Слишком тёмно → добавить белый
- da > 0: Слишком красно → добавить da < 0 (зелёный/синий)
- da < 0: Слишком зелено → добавить da > 0 (красный)
- db > 0: Слишком жёлто → добавить db < 0 (синий)
- db < 0: Слишком сине → добавить db > 0 (жёлтый)

### Остаточная дельта
После корректировки остаточные дельты должны быть < 0.3 (в допуске)

## Troubleshooting

### Тесты не запускаются
1. Убедитесь, что `test.js` подключён в `index.html`
2. Проверьте консоль на ошибки (F12)
3. Убедитесь, что `app.js` загружен раньше `test.js`

### Тесты провалены
1. Смотрите детали в консоли (expected vs actual)
2. Проверьте входные данные формулы
3. Убедитесь, что база пигментов загружена
